# Task 1: To-Do List Application

## Description

A console-based To-Do List application that allows users to manage daily tasks efficiently.

Users can add tasks, view existing tasks, mark tasks as completed, and delete tasks. The application stores task data using JSON file handling so tasks remain available even after closing the program.

## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Automatic task ID generation
- Save and load tasks using JSON file storage
- User-friendly menu interface

## Concepts Used

- Python Lists
- Dictionaries
- Functions
- Loops
- Conditional Statements
- File Handling
- JSON Data Storage
- Input Validation

# Task 3: Random Password Generator

## Description

A secure password generator application that creates strong random passwords according to user preferences.
The program allows users to select password length and choose whether to include uppercase letters, numbers, and special characters.
It uses Python's `secrets` module to generate cryptographically secure random passwords.

## Features

- Custom password length selection
- Uppercase letter support
- Number inclusion option
- Special character support
- Password strength evaluation
- Secure random password generation

## Concepts Used

- Python String Module
- Secrets Module
- Functions
- Loops
- Lists
- Random Character Selection
- Input Validation
- Secure Programming Practices
